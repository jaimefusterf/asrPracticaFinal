Instrucciones de la prueba

1. Inserta aquí tu URL

Al insertar la URL de una imagen, el motor de reconocimiento visual identificará lo que está representado en la imagen y le dirá en inglés como si estuvieras jugando contra tu computadora.

2. Tradúcelo

Escribe una palabra, se traducirá al español gracias al Microservicio de traducción

3. Escúchalo

Descubre cómo se pronuncia.

4. Mira lo que has guardado

Además, la palabra se guardará en una base de datos para que luego puedas revisar todo tu nuevo vocabulario en español.

Páctica Final:
Microservicios IBM Cloud

En esta práctica final se ha implementado una aplicación mediante una serie de microservicios. Esta se basa en un método para aprender español en el que el usuario podrá introducir una imagen y descubrirá lo representado en la misma en español, traducido en texto y audio. Para ello, se han implementado los siguientes servicios:

Visual Recognition
Language Translator
Text to Speech
Cloudant
